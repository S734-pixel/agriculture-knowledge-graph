# 🌱 Agriculture Knowledge Graph for Smart Farming

## 📌 Overview
This project is part of my MSc in Artificial Intelligence at Ulster University.

It focuses on building a semantic knowledge graph using real-world agricultural datasets.

## 🚀 Features
- Ontology design using AGROVOC and SSN/SOSA
- RDF data transformation
- SPARQL queries for intelligent insights
- Integration concept for IoT sensor data

## 🛠️ Technologies Used
- Python
- Protege
- RDF
- SPARQL

## 📊 Dataset
Smart Farming 2024 Dataset (Kaggle)
